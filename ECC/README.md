# ECC — 에러코드 원본 보관

이 폴더는 에러코드 원본 자료(엑셀, 매뉴얼 스크랩, PDF 등)를 보관하는 공간입니다.

## 용도
- `errors_v2.json` 생성의 **원천 자료** 보관
- 제조사별 서비스 매뉴얼에서 추출한 엑셀 파일 등
- 신규 모델 추가 시 원본 참고자료

## 동기화
- NAS: `\\192.168.0.249\ErrorCode\ECC\`
- GitHub: 이 폴더
- 두 위치는 항상 동일 상태로 유지됩니다.

## 주의
- **Public 저장소**이므로 여기에 두는 파일은 누구나 볼 수 있습니다.
- 영업 기밀성 자료는 NAS에만 두고 이 폴더에는 복사하지 마세요.
